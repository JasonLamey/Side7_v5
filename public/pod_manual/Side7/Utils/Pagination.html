NAME

Side7::Utils::Pagination;

DESCRIPTION

Supplies tools and functionality for generating necessary data for pagination menus.

FUNCTIONS

get_pagination()

    $pagination = Side7::Utils::Pagination::get_pagination( { 
                                    total_count => $total, 
                                    page => $page,
                                    pagination_limit => $pagination_limit 
                            } );

Returns a hashref of pagination variables, including previous page number, next page number, total count, first item, last item, total pages. pagination_limit is optional, and defaults to the configured amount.
