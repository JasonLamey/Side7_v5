NAME

Side7::Login

DESCRIPTION

This class manages the login/logout status of a user.

FUNCTIONS

user_login

    my ( $redirect_url, $user_object ) = Side7::Login::user_login(
                        username => $username,
                        password => $password,
                        rd_url   => $rd_url,
                    );

Logs a user in. If the username and password are valid, the user is logged in. If not, the login fails. Returns a redirect path, and the user object if the user is logged in.

is_logged_in

    $is_logged_in = Side7::Login->is_logged_in();

Returns the user's session cookie if the user is logged in. Redirects to the index page otherwise.

sanitize_redirect_url

    my $rd_url = Side7::Login::sanitize_redirect_url(
        { rd_url => params->{'rd_url'}, referer => request->referer, uri_base => request->uri_base }
    );

Cleans up any redirect URL intended to be passed to the login_form, and ensures that (a) it's not from outside the site, and (b) it's cleaned up and has the domain removed.
